# 🚗 Modern Autókölcsönző Weboldal

Egy modern, reszponzív autókölcsönző rendszer.


### Funkciók
- **Regisztráció és bejelentkezés** - Biztonságos autentikáció telefonszámmal
- **Autó keresés és szűrés** - Keresés márka, kategória, üzemanyag, váltó, ülések száma és ár alapján
- **Autó részletek** - Részletes információk, képek, jellemzők
- **Foglalás készítése** - Dátum választás automatikus árkalkulációval
- **Saját foglalások** - Korábbi és aktív bérlések megtekintése
- **Profil kezelés** - Személyes adatok megtekintése és módosítása
- **Dashboard** - Statisztikák (autók, felhasználók, aktív bérlések, összes bevétel)
- **Autó menedzsment** - Új autók hozzáadása, szerkesztés, törlés
- **Felhasználó kezelés** - Admin jogosultságok adása/visszavonása
- **Bérlések áttekintése** - Legutóbbi bérlések listája képekkel
